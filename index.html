About


Independent Developer graduated in Electrical Engineer with major in Control and Automation, while had experience in microprocessors, embedded programming, robotics, GIS systems and data analysis, also gained knowledge on Unity engine. Since 2016 on the Asset Store, with more than 8 years of experience in Unity providing tools to help game developpers build Creative Gameplays, improve the User Experience and build STEM (Science, Technology, Engineering and Mathematics) games and providing after-sale technical support on using the assets.

Main project: The Play Mode Blocks Engine is a complete project that implements a blocks programming Engine that can be used to develop STEM educational apps focused on code learning and a variety of puzzle games.
